Design a complete flagship frontend for a product called **RepoSage**.

RepoSage is a premium code intelligence platform that helps developers understand unfamiliar codebases through architecture mapping, dependency graphs, file/module exploration, onboarding docs, insights, and grounded AI-assisted explanation.

This is not a consumer app.
This is not a generic AI SaaS.
This is not a dashboard template.
This is not a startup boilerplate website.

The design must feel:
- handcrafted
- premium
- smooth
- minimal
- highly intentional
- technically elegant
- modern
- memorable
- visually intelligent
- calm but impressive
- like a high-end product made by a top UI/UX design team

The design must NOT feel:
- AI-generated
- template-based
- repetitive
- generic SaaS
- copied from a dashboard kit
- too corporate
- too startup-marketing-like
- too crowded
- too noisy
- too text-heavy
- over-layered
- over-glassy
- over-glowy
- gimmicky
- cheap futuristic
- “look how modern this is” fake design

We want this to feel like a serious, premium, pro-level developer product.

1. PRODUCT CONTEXT
==================================================

RepoSage helps developers:
- upload or connect a repository
- analyze code structure
- visualize architecture and dependencies
- inspect files and modules
- see risky or high-impact files
- understand likely entry points
- explore reading paths
- ask AI questions grounded in real code evidence
- read generated architecture/onboarding docs
- eventually support startup-level workflows like impact analysis and code review intelligence

This is a product about:
- systems
- structure
- architecture
- graphs
- code relationships
- depth
- clarity
- intelligence
- precision

The visual language should reflect that.
